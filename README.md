## 工夫した点

- タブで情報を切り替えられるようにした
- 小樽、プログラミング、介護経験を画像カルーセルで表現した
- トップ画像とカルーセル画像に、ゆっくり自然に動くアニメーションとなめらかな切り替えを加えた
- スマートフォンでは画像の動きを控えめにして、見やすさを優先した
- 画像アニメーションのカクつきを抑えるため、波や光の動きを軽い処理に調整した
- 文字、画像キャプション、矢印、ドットに軽いアニメーションを追加した
- トップ画面の登場演出、光の通過、画像切り替え時のズーム演出で印象を強めた
- トップに制作物と介護経験への導線を追加し、プロフィールの目的を伝わりやすくした
- ポートフォリオを「作ったもの」「できるようになったこと」「作っているもの」「次に作るもの」に整理した
- 最後に目指す方向性を一文で伝える締めのメッセージを追加した
- スマートフォンでも読みやすいように余白やボタンサイズを調整した
- 個人の連絡先やSNSは掲載せず、公開目的を明確にした
- 公開ページを共有しやすいようにURLコピー機能を追加した
- 学習成果への導線を強くし、上部からポートフォリオへ移動しやすくした
- 学習成果欄に「公開済み」「改善中」「学習中」の要約を追加した
- 公開ページに出さない確認済み項目をREADMEへ移した
- プロフィール文章を短くし、初めて見る人にも読みやすい表現に整えた
- 経歴・歩みを、仕事の経験と学んだことが伝わる形に整理した
- 好きなこと欄に導入文とラベルを追加し、人柄が伝わりやすい形にした
- 画像キャプションをプロフィールの流れに合う自然な表現に調整した
- 別アプリの大きな紹介は外し、プロフィール作成の内容に絞った
- トップに一言プロフィールを追加し、初めて見る人にも人物像が伝わりやすい形にした
- 経歴・歩みの最後に、これまでの経験が今のWeb制作学習へつながるまとめを追加した
- 好きなこと欄に小さな文字アイコンを追加し、文章だけにならない見せ方にした
- ページ下部にプロフィール全体で伝えたいことを短くまとめた
- 文章の行間と余白を広げ、高齢者の方にも読みやすい表示に調整した
- ボタンとタブの高さ、余白、間隔を広げて押しやすくした
- 画像キャプション、表、公開情報の余白を広げて確認しやすくした
- 公開についての説明文を、安心して読めるやさしい表現に整えた

## 自分用チェックメモ

画面上には表示しない確認用メモです。

- PCで読みやすい余白になっている
- スマートフォンでタブとボタンが押しやすい
- 画像が自動で切り替わる
- 個人の連絡先やSNSを公開していない
- 学習成果への導線が上部と学習タブ内にある
- プロフィール作成以外のアプリ内容が前面に出すぎていない
- 経歴、自己紹介、好きなことが初心者にも読みやすい文章になっている
- トップを見て、3秒以内に人物像が伝わる
- 経歴の最後で、介護経験とWeb制作学習のつながりが伝わる
- 好きなこと欄が文章だけで単調になっていない
- ページ下部で、プロフィール全体の印象がもう一度まとまっている
- 文章の行間が詰まりすぎていない
- タブやボタンがスマートフォンでも押しやすい高さになっている
- 表の項目が読みやすい余白になっている
- 画像キャプションが小さく見えづらくなっていない
- 公開情報の説明がやさしく、誤解されにくい表現になっている

## 今後の改善予定

- 文章や画像の見え方を、公開後の確認に合わせて調整する
- プロフィール写真や小樽画像の見え方を必要に応じて調整する
- 学習成果が増えたら、プロフィールサイト内に自然な形で追記する
