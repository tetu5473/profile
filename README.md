## 工夫した点

- タブで情報を切り替えられるようにした
- 小樽、プログラミング、介護経験を画像カルーセルで表現した
- トップ画像とカルーセル画像に、ゆっくり自然に動くアニメーションを加えた
- スマートフォンでも読みやすいように余白やボタンサイズを調整した
- PC・スマートフォンで見やすいことが伝わる確認欄を追加した
- 個人の連絡先やSNSは掲載せず、公開目的を明確にした
- 公開ページを共有しやすいようにURLコピー機能を追加した
- 太田の習慣コーチアプリを、目的・機能・学んだことが伝わる形で追加した
- 太田の習慣コーチアプリの画面画像を追加し、公開URLは準備中として自然に表示した
- 太田の習慣コーチアプリの画像を、スマホ画面風の見せ方と自然な流れのアニメーションに調整した
