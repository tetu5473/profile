# Tetsuya Ota Profile

太田鉄也のプロフィールサイトです。

北海道小樽市でのルーツ、介護の現場での経験、Web制作とプログラミング学習への挑戦をまとめています。

## 公開URL

https://tetu5473.github.io/profile/

## このサイトで伝えたいこと

## 工夫した点

- タブで情報を切り替えられるようにした
- 小樽、プログラミング、介護経験を画像カルーセルで表現した
- トップ画像とカルーセル画像に、ゆっくり自然に動くアニメーションとなめらかな切り替えを加えた
- スマートフォンでは画像の動きを控えめにして、見やすさを優先した
- 画像アニメーションのカクつきを抑えるため、波や光の動きを軽い処理に調整した
- 文字、画像キャプション、矢印、ドットに軽いアニメーションを追加した
- トップ画面の登場演出、光の通過、画像切り替え時のズーム演出で印象を強めた
- トップに制作物と介護経験への導線を追加し、プロフィールの目的を伝わりやすくした
- ポートフォリオを「作ったもの」「できるようになったこと」「作っているもの」「次に作るもの」に整理した
- 最後に目指す方向性を一文で伝える締めのメッセージを追加した
- スマートフォンでも読みやすいように余白やボタンサイズを調整した
- 個人の連絡先やSNSは掲載せず、公開目的を明確にした
- 公開ページを共有しやすいようにURLコピー機能を追加した
- 太田の習慣コーチアプリを、目的・機能・学んだことが伝わる形で追加した
- 太田の習慣コーチアプリの画面画像を追加し、公開URLは準備中として自然に表示した
- 太田の習慣コーチアプリの画像を、スマホ画面風の見せ方と自然な流れのアニメーションに調整した
- 制作物への導線を強くし、上部からポートフォリオへ移動しやすくした
- 制作物欄に「公開済み」「制作中」「学習中」の要約を追加した
- 公開ページに出さない確認済み項目をREADMEへ移した

## 自分用チェックメモ

画面上には表示しない確認用メモです。

- PCで読みやすい余白になっている
- スマートフォンでタブとボタンが押しやすい
- 画像が自動で切り替わる
- 個人の連絡先やSNSを公開していない
- 制作物への導線が上部と学習タブ内にある
