自己紹介

介護の現場で培った「相手の立場で考える力」を、Web制作の学習にも活かしていきたいです。

はじめまして。太田鉄也です。北海道小樽市出身で、大原簿記専門学校 札幌校を卒業しました。 小樽で過ごした時間は、今の自分の土台になっています。

学生時代は小学校3年生から高校3年生まで野球を続けました。 続けること、チームで動くこと、最後まで諦めないことを学びました。

その後は介護の仕事に進み、約23年間、利用者さんやご家族と向き合ってきました。 相手が何に困っているのかを考え、安心してもらえる声かけを大切にしてきました。

現在はWeb制作を学んでいます。 介護経験で身につけた気配りや観察力を、見やすく迷わない画面作りに活かしていきたいです。

これまで人と向き合ってきた経験を大切にしながら、見る人にとって伝わりやすいプロフィール作りを続けていきたいです。

介護職 介護の仕事には、トータルで約23年間携わっています。デイケアサービスで約5年間勤務した後、 介護福祉士としてグループホーム、小規模多機能型施設、老人保健施設などで経験を積みました。
ケアマネ 介護支援専門員に合格後、約10年間ケアマネジャーとして従事しています。 現場と相談支援の両方を経験し、人の状況を整理して伝える大切さを学んできました。
仕事で学んだこと 利用者さんから「ありがとう」と言われたときに、大きなやりがいを感じてきました。 認知症の方と関わる中では、こちらの考えを押し付けず、相手の気持ちを想像して対応することを大切にしてきました。
強み 相手の立場で物事を考え、気配りをしながら行動することです。 レクリエーションや送迎、声かけの場面でも、安心してもらえる関わりを意識してきました。
転機 40代に入り、これからの人生を考える中で「今のままで本当にいいのか」と感じるようになりました。 Web制作の学習を始め、これまでの経験を新しい形で活かすことに挑戦しています。
野球経験 小学校3年生から高校3年生まで野球を経験。高校時代は硬式野球部で、ポジションはファースト。

これまでの歩み

  1. 小樽で育ち、野球に打ち込む 学生時代は野球を通して、継続することや仲間と頑張ることを学びました。
  2. 専門学校で資格取得に取り組む 簿記や電卓、公務員試験の勉強に取り組み、学ぶことの大切さを経験しました。
  3. 介護の現場で23年、人と向き合う 介護職・介護福祉士・介護支援専門員として、利用者さんやご家族と関わり、人の立場に立つことを学びました。
  4. 40代からプログラミングに挑戦 このままでいいのかという思いから、人生を変えるために新しい分野へ挑戦しています。
  5. 経験をWeb制作に活かす 相手の立場で考える力を、見やすく迷わないページ作りに活かしていきたいです。
今につながっていること

野球で学んだ継続力、介護の現場で身につけた気配り、ケアマネジャーとしての整理して伝える力。 これらを土台に、今は「見る人に伝わるプロフィール作り」と「使いやすい画面作り」を学んでいます。

Web制作を学ぶ理由

将来の選択肢を広げ、これまでの経験を新しい形で活かすために、Web制作を学んでいます。

介護の仕事を長く続ける中で、これからの働き方や成長について考えるようになりました。 その中で、Web制作を学び、自分の経験を新しい形で活かしたいと思うようになりました。

ただ技術を覚えるだけではなく、見る人にとってわかりやすく、 迷わず読めるページを作れる人を目指しています。

介護の現場で学んだ「相手の状況を想像する力」を、 文章の整理や画面の導線作りにも活かしていきたいです。

学習しているスキル

Webページがどのように作られているのかを理解するために、まずは基本となるHTML、CSS、JavaScriptを中心に学んでいます。 まだ学習中ですが、少しずつ「読むだけ」ではなく「自分で作って試す」ことを大切にしています。

HTML

文章、見出し、画像、表など、ページの内容を組み立てるための言語として学んでいます。

CSS

色、余白、文字の大きさ、スマートフォン対応など、見やすい画面に整える方法を学んでいます。

JavaScript

タブの切り替えやボタン操作など、ページに動きをつけるための基本を学んでいます。

Git・GitHub

作ったファイルを記録し、変更の履歴を残しながら制作を進める方法を学んでいます。

このサイトを作った理由

このサイトは、Web制作の学習成果として、自分のことをわかりやすく伝えるために作りました。

初めて見る人にも、自分がどのような経験をしてきたのか、今なぜプログラミングを学んでいるのかが伝わるように意識しました。 文章をただ並べるだけではなく、プロフィール、経歴、学習目標、好きなことに分けることで、読みたい情報を探しやすくしています。

工夫した点は、介護の経験とこれから学びたいITの分野をつなげて説明したことです。 「人と向き合ってきた経験」を、今後は「使う人にとってわかりやすい画面作り」に活かしていきたいと考えています。
